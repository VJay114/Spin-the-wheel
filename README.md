## ✨ Features

| Feature | Description |
|---|---|
| ⚡ Momentum Spinning | Press `Space` or click **Spin!** repeatedly to build up speed |
| 🖼️ Avatar Images | Add image URLs per participant — shown on wheel segments & winner popup |
| 🏆 Winner Modal | Popup announcement with name and photo when the wheel stops |
| 📋 Participant Table | Clean editable table with **Name** and **Image URL** columns |
| 📊 Speed Bar | Visual indicator showing current spin velocity |
| 🕓 Spin History | Tracks the last 8 winners with photos |
| 🎨 Image Toggle | Switch between name-only and image mode instantly |
| 📁 Single File | Everything in one `index.html` — no install, no build step |

## 🕹️ How to Use

1. **Add participants** — Fill in names in the table on the right panel
2. **Add images** *(optional)* — Toggle **Show Images** on, then paste any image URL in the Image URL column
3. **Spin the wheel:**
   - Press `Space` to add speed — **the more you press, the faster it spins!**
   - Or click the **Spin!** button / the center cap of the wheel
4. **View the winner** — A popup announces the winner when the wheel stops
5. Press `Space` or **Spin Again** to dismiss and go another round

### ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `Space` | Add spin speed / dismiss winner modal |
| `Esc` | Close winner modal |
