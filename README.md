## рҹ•№пёҸ How to Use

1. **Add participants** вҖ” Fill in names in the table on the right panel
2. **Add images** *(optional)* вҖ” Toggle **Show Images** on, then paste any image URL in the Image URL column
3. **Spin the wheel:**
   - Press `Space` to add speed вҖ” **the more you press, the faster it spins!**
   - Or click the **Spin!** button / the center cap of the wheel
4. **View the winner** вҖ” A popup announces the winner when the wheel stops
5. Press `Space` or **Spin Again** to dismiss and go another round

### вҢЁпёҸ Keyboard Shortcuts

| Key | Action |
|---|---|
| `Space` | Add spin speed / dismiss winner modal |
| `Esc` | Close winner modal |
