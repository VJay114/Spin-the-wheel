🎡 Spin the Wheel
A lightweight, single-file spin wheel app built with vanilla HTML, CSS, and JavaScript. Perfect for random name picking, classroom activities, giveaways, or any group decision-making.
Features

Momentum-based spinning — press Space or click Spin repeatedly to build up speed
Participant table with Name and Image URL columns
Optional avatar images shown on wheel segments and winner popup
Winner modal with photo support
Spin history log
Speed indicator bar
Zero dependencies — just one index.html file

Usage

Download or clone the repo
Open index.html in any browser
Add participant names (and optional image URLs) in the table
Press Space or click Spin! to spin — the more you press, the faster it goes!

Built With
